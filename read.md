# Video Chat Uygulaması

WhatsApp benzeri gerçek zamanlı video sohbet uygulaması.

## Özellikler
- ✅ Gerçek zamanlı sohbet
- ✅ Video paylaşımı
- ✅ YouTube entegrasyonu
- ✅ Görüntülü/sesli arama
- ✅ Dosya paylaşımı
- ✅ Oda sistemi

## APK Yapma
1. Bu repoyu GitHub'a yükle
2. https://appgenerator.org sitesine git
3. GitHub URL'sini yapıştır
4. APK'yı indir!
